Module 5: Loops 🔁

🔁 What is a Loop? 🔁

Imagine you are standing at the production line and need to check every single bag of popcorn as it passes by. Bag 1, bag 2, bag 3 ... if there are 100 bags, you would have to write 100 separate lines of code! That would take forever.

A loop is a special instruction that tells the computer to repeat a task over and over — as many times as you need. Instead of writing the same code 100 times, you write it once inside a loop and the computer does the rest!

Loops Are Everywhere

You already use loops in real life without thinking about it:

  • Brushing your teeth — you brush back and forth many times
  • Counting children in class — "1, 2, 3, 4, 5 ..."
  • Stacking boxes — you pick up one box, stack it, and repeat
  • The factory line — items move past and get counted one by one

In coding, a loop does exactly the same thing — it repeats an action a specific number of times so you do not have to write it out over and over!

💡 Remember: Loops save time and reduce mistakes. If you need to do something more than once, a loop is almost always the answer!

🏗️ The Structure of a FOR Loop 🏗️

The most common type of loop for counting is called a FOR loop. It has three parts that work together to control how many times the loop runs.

The Three Parts:

  • Where to start — the number you begin counting from
  • Where to stop — the number you count up to
  • The counter — a variable that tracks which repetition you are on

Here is how we write a FOR loop in pseudocode:

FOR "item number" FROM 1 TO 5 DO
  PRINT "Checking item", "item number"
END FOR

What this does:

  • Starts counting at 1
  • Runs the code inside the loop
  • Adds 1 to the counter automatically
  • Repeats until the counter reaches 5

Output:

Checking item 1
Checking item 2
Checking item 3
Checking item 4
Checking item 5

Five lines of output from just three lines of code! That is the power of a loop.

💡 Remember: The FOR loop automatically adds 1 to the counter each time. You do not need to tell it to count up — it does that for you!

🔢 Using the Counter Variable 🔢

The counter variable inside a loop is incredibly useful. It keeps track of which repetition you are on — and you can use it in your code!

The Counter Changes Automatically

Each time the loop runs, the counter increases by 1 on its own:

FOR "day number" FROM 1 TO 5 DO
  PRINT "Day", "day number"
END FOR

Output:

Day 1
Day 2
Day 3
Day 4
Day 5

You Can Use the Counter in Calculations

Because the counter is just a variable, you can use it to calculate other things. For example, if each day you earn points equal to the day number:

FOR "day number" FROM 1 TO 3 DO
  SET "points earned" = "day number" * 10
  PRINT "Day", "day number", "— earned", "points earned", "points"
END FOR

Output:

Day 1 — earned 10 points
Day 2 — earned 20 points
Day 3 — earned 30 points

💡 Remember: The counter variable is the heart of a loop. Use it to number things, calculate values, or track progress!

📊 Tracking Totals with Loops 📊

One of the most powerful things you can do with a loop is keep a running total. You create a variable before the loop starts, then add to it inside the loop each time it runs.

How It Works:

SET "total points" = 0
FOR "day number" FROM 1 TO 5 DO
  SET "total points" = "total points" + 10
  PRINT "Day", "day number", "— Total:", "total points"
END FOR

Output:

Day 1 — Total: 10
Day 2 — Total: 20
Day 3 — Total: 30
Day 4 — Total: 40
Day 5 — Total: 50

The Pattern:

  • Before the loop: create the total variable and set it to 0
  • Inside the loop: add to the total each time
  • After the loop: the total variable holds the final answer

This is how the factory tracks inventory — counting items one by one and keeping a running total of everything that passes through!
