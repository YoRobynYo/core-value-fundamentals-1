Module 5: Loops Part 2 🔢

🔢 Loops Can Do Maths! 🔢

In the last part, you learned how to make a loop count numbers. But loops can do much more than just count! You can put any calculation inside a loop — addition, subtraction, multiplication, or division. The loop will perform that calculation for each cycle, making it easy to manage supplies for the whole factory!

How It Works

Imagine you have 10 machines, and each machine needs 5 bags of corn. Instead of writing the same calculation 10 times, you can write one loop that runs 10 times. Inside the loop, you tell the computer: "Calculate the total bags for this machine number and print it." The loop does the hard work for you!

FOR "machine number" FROM 1 TO 10 DO
  SET "total bags" = "machine number" * 5
  PRINT "Machine", "machine number", "needs", "total bags", "bags"
END FOR

Output:
Machine 1 needs 5 bags
Machine 2 needs 10 bags
Machine 3 needs 15 bags
... and so on up to Machine 10.

💡 Remember: Inside the loop, the counter (like "machine number") changes automatically each time. You can use it in any calculation to create a different result for each cycle!
