## TEACHING PHILOSOPHY — READ THIS FIRST

The goal of Part 2 is to make RETURN feel completely natural.
Children already know the function shape from Part 1.
Now the function hands something back instead of just printing.

- Exercises 1-2: RETURN with no parameters — just the new concept
- Exercises 3-4: RETURN with a parameter — something goes in, something comes back
- Exercise 5: Outside variable + parameter + RETURN — the full tree

Every exercise uses familiar classroom supply variables from Module 2.
No new concepts other than RETURN.

## EXERCISE 1: The Pencil Colour ✏️

**SECTION NAME:** functions
**EXERCISE NUMBER:** 1
**EMOJI:** ✏️

**SCENARIO:** The classroom needs to know what colour the pencils are. Write a function that returns the colour!

**VARIABLES TO USE:**
- Part A: no parameters — RETURN a fixed value
- Part B: no parameters — RETURN a different fixed value

**CONCEPT:** Simplest RETURN — no parameters, just a function that hands one value back.

**PART A FOCUS:** Write a function called "get pencil colour" that returns "yellow". Catch the return value in a variable called "pencil colour" and print it.
**PART B FOCUS:** Change the function to return "red". Catch and print the same way.

**EXPECTED OUTPUT PART A:**
```
The pencil colour is yellow
```
**EXPECTED OUTPUT PART B:**
```
The pencil colour is red
```

## EXERCISE 5: The Full Supply Check 🌟

**SECTION NAME:** functions
**EXERCISE NUMBER:** 5
**EMOJI:** 🌟

**SCENARIO:** The full classroom supply check — classroom name from outside, supply name as a parameter, returned and printed together!

**VARIABLES TO USE:**
- Part A: "classroom name" outside the function. "supply name" as a parameter.
- Part B: "classroom name" outside the function. "supply name" as a parameter.

**CONCEPT:** Outside variable + parameter + RETURN — the full tree.

**PART A FOCUS:** Set "classroom name" to "Room 3" outside the function. Write a function called "check supply" that takes "supply name" as a parameter and returns it. Catch the return in "checked supply" and print both "classroom name" and "checked supply". Call with "pencil" then "ruler".
**PART B FOCUS:** Change "classroom name" to "Room 5". Call the same function with "scissors" then "glue stick".

**EXPECTED OUTPUT PART A:**
```
Classroom: Room 3
Supply checked: pencil
Classroom: Room 3
Supply checked: ruler
```
**EXPECTED OUTPUT PART B:**
```
Classroom: Room 5
Supply checked: scissors
Classroom: Room 5
Supply checked: glue stick
```

**Opening for Module 7 Part 2:**
"You already know how to write a function and call it. Now your functions are going to do something even more powerful — they are going to hand information back to you!
