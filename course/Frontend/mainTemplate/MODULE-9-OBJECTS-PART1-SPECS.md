# MODULE 9: OBJECTS — SPECIFICATIONS

## THEME: Bead Box 🎨

In this module, you'll learn about **objects** — a way to group related information together. Think of an object like a bead box, where each bead has properties like colour, shape, and size.

---

## EXERCISE 1: Spot the Properties 🔍

**Scenario:** Look at this bead object and answer the questions.

```
SET "magic bead" = {
  "colour": "blue",
  "shape": "star",
  "glow": "yes"
}
```

**Part A:** What is the "colour" of the magic bead?

- Use PRINT and dot notation to display the "colour" property of "magic bead"
- Expected output: `blue`

**Part B:** What is the "shape" of the magic bead?

- Use PRINT and dot notation to display the "shape" property of "magic bead"
- Expected output: `star`

---

## EXERCISE 2: Build Your Bead Object 🔨

**Scenario:** Now you create your own bead!

**Part A:** Create an object called "my bead" with these three properties:
- "colour": "red"
- "shape": "circle"
- "size": "small"

```
SET "my bead" = {
  "colour": "red",
  "shape": "circle",
  "size": "small"
}
```

**Part B:** Now read back each property using PRINT and dot notation to check it was created correctly.

- PRINT the "colour" — expected output: `red`
- PRINT the "shape" — expected output: `circle`
- PRINT the "size" — expected output: `small`

---

## EXERCISE 3: Read Your Bead 📖

**Scenario:** Using the "my bead" object from Exercise 2, read its properties.

**Part A:** Use PRINT and dot notation to display the "colour" of "my bead".

- Expected output: `red`

**Part B:** Use PRINT and dot notation to display the "shape" of "my bead".

- Expected output: `circle`

---

## EXERCISE 4: Update a Property ✏️

**Scenario:** Your bead has been upgraded!

**Part A:** Change the "colour" of "my bead" from "red" to "purple" using SET and dot notation.

```
SET "my bead"."colour" = "purple"
```

Then PRINT the "colour" to confirm the change. Expected output: `purple`

**Part B:** Change the "size" of "my bead" from "small" to "medium" using SET and dot notation.

```
SET "my bead"."size" = "medium"
```

Then PRINT the "size" to confirm the change. Expected output: `medium`
