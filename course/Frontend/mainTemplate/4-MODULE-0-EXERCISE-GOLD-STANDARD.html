Exercise 1: Classroom Introduction 👋

Welcome to our classroom! Before we learn about Fundamentals, let's get organised. The teacher is creating a classroom chart and needs everyone's information. Let's use variables to store where everyone sits!

Part A: Introduce Yourself

Let's store your information so everyone knows who you are! We need to know your name, which desk you're sitting at, and which classroom we're in today.

  • Step 1:  Create a variable named "child name" and set its value to "Ben" (or use your own name!).
  • Step 2:  Create a variable named "desk number" and set its value to 7.
  • Step 3:  Create a variable named "classroom number" and set its value to 102.
  • Step 4:  Now write the code to introduce yourself:
    SET "child name" = "Ben"
    SET "desk number" = 7
    SET "classroom number" = 102
    PRINT "My name is ", "child name"
    PRINT "I sit at desk ", "desk number", " in classroom ", "classroom number"
