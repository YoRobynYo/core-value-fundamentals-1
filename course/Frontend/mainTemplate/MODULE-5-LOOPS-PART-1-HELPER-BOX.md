## NARRATIVE CONTEXT

**Opening for Module 5, Part 1:**
"Welcome back! In the classroom and the supply room, some jobs are very repetitive, like counting every box on a shelf or giving every child a sticker. Instead of doing these tasks one by one, we can use a LOOP to do them automatically! A loop is a special instruction that tells the computer to repeat a task as many times as we want. Let's learn how to save time with loops!"

**Exercise Flow:**
1. Simple counting — checking a row of pencil boxes.
2. Tracking a total — adding up homework points over a week.
3. Addition in a loop — calculating bonus stickers for each child.
4. Multiplication in a loop — working out how many pencils each desk needs.
5. A bigger challenge — organising hundreds of pencils into labelled boxes.

**Closing:**
"Amazing work! You've learned how to create loops to repeat tasks, track totals, and perform calculations automatically. You can now handle jobs that would take forever to do by hand! In the next part, we'll explore how to make our loops even more powerful by adding decisions inside them."
