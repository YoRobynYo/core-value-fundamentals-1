## NARRATIVE CONTEXT

**Opening for Module 5, Part 3:**
"You're getting amazing at using loops! You've learned to count up, add, and multiply. Now it's time to learn how loops can help us count down and share things out. 1. Simple subtraction — tracking oil containers as they are used one by one.

4. Division in a new context — organising bags onto shelves.
