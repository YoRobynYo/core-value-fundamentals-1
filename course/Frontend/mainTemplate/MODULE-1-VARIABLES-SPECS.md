## EXERCISE 1: CLASSROOM INTRODUCTION 👋

**SECTION NAME:** variables
**EXERCISE NUMBER:** 1
**EMOJI:** 👋

**SCENARIO:** A child introduces themselves with their personal classroom information

**VARIABLES TO USE:**
- Part A: "child name", "desk number", "classroom number"
- Part B: "teacher name", "subject", "lesson number"

**CONCEPT:** Creating variables to store and display personal information

**PART A FOCUS:** Store the child's personal classroom information — name, desk, classroom
**PART B FOCUS:** Store teacher information and lesson details

**EXPECTED OUTPUT PART A:** "My name is Ben. I sit at desk 7 in classroom 102."
**EXPECTED OUTPUT PART B:** "My teacher is Mrs Jones and this is Fundamentals lesson 1."

**VARIABLES TO USE:**

## NARRATIVE CONTEXT

**Opening for Module 1:**
"Welcome to our classroom! Before we can start learning, we need to get organised.
Just like we label our desks, cubbies, and supplies — in coding we use VARIABLES to
label and store information. Let's start by getting to know each other and mapping
our classroom!"

**Exercise Flow:**
1. Personal introduction — who am I, where do I sit?
2. Supply cupboard mapping — where are the supplies?
3. Classroom directory — getting to know our classmates
4. Teacher information — who teaches us and what?
5. Daily timetable — organising our time

**Closing:**
"Great work! You've learned how to store and organise information using variables.
Next we'll use these skills to COUNT things and do CALCULATIONS with operators.
But first — guess what? We're going on a field trip to the Popcorn Factory! 🍿"

## QUALITY CHECKLIST FOR AI GENERATION

When AI generates the exercise, verify:
- [ ] Section name is "variables"
- [ ] Exercise number matches
- [ ] Both Part A and Part B are included
- [ ] Variables use natural language in quotes — NOT camelCase
- [ ] Uses pseudocode only — SET and PRINT — no real language syntax
- [ ] Uses PRINT with commas — no + concatenation
- [ ] Uses `<code>` tags (not `<highlight>`)
- [ ] IDs follow pattern: variables-ex[number]-part-[letter]
- [ ] Data attributes correct: data-section="variables"
- [ ] Empty hint/output/answer containers present
- [ ] Exercise 1 Part A and Part B show complete pseudocode answer
- [ ] Exercises 2-5 Part A and Part B reference specific task — no pseudocode shown
- [ ] Expected outputs match the spec
- [ ] British English throughout — lessons not periods, colour not color
- [ ] Nothing used that hasn't been taught in a helper box (Rule 6)
