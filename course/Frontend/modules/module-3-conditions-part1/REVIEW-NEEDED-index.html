  • Step 1:  Create a variable named "child name" and set its value to "Ben" (or use your own name!).
  • Step 2:  Create a variable named "desk number" and set its value to 7.
  • Step 3:  Create a variable named "classroom number" and set its value to 102.

    SET "desk number" = 7
