Exercise 1: COUNTING BAGS ON THE PRODUCTION LINE 📦

Part A: Counting the First 5 Bags

A child needs to count bags on a production line as they pass by. The child must keep track of the "number of bags" that have been counted so far. To start, the child will count the first 5 bags on the production line. The child will use a coding technique to make this task easier and more efficient.

  • Step 1:  Create a variable named "number of bags" and set its value to 0.
  • Step 2:  Create a variable named "number of bags" and set its value to 0, then use a loop to count the bags one by one, starting from 1 and going up to 5.
  • Step 3:  Use PRINT to display the message "Bag X is ready" for each bag, where X is the current "number of bags" being counted.
  • Step 4:  Now write the code to set up the counting process:
    SET number of bags = 0
    SET number of bags = number of bags + 1
    PRINT "Bag ", number of bags, " is ready"
    REPEAT FOR number of bags FROM 1 TO 5
