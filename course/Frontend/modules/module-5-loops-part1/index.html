Opening for Module 5, Part 1:

"Welcome back! In the classroom and the supply room, some jobs are very repetitive, like counting every box on a shelf or giving every child a sticker. Instead of doing these tasks one by one, we can use a LOOP to do them automatically! A loop is a special instruction that tells the computer to repeat a task as many times as we want. Let's learn how to save time with loops!"

1. Simple counting — checking a row of pencil boxes.

2. Tracking a total — adding up homework points over a week.
